# Memórias

O **memorias** é um projeto web desenvolvido em React com Vite, pelos alunos do 8º ano da Escola Municipal Abel Freire de Aragão em Campo Grande, MS.

---

## Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes ferramentas:

* **React** (com Vite)
* **Node.js**
* **React Router DOM** (para navegação entre páginas)
* **React Icons** (para os ícones da interface)

---

## Como Rodar o Projeto na sua Máquina

Siga o passo a passo abaixo para clonar e executar o projeto localmente:

### 1. Pré-requisitos
Antes de começar, você precisa ter o **Node.js** instalado. No Linux (Ubuntu/Debian), instale rodando:
```bash
sudo apt update
sudo apt install nodejs npm
npm install react-router-dom
npm install react-icons
